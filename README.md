# Quora

Quora (https://www.quora.com/) is a Q&A platform where questions are asked, answered, edited, and organized by its community of users. Founded in 2009, Quora connects people who have knowledge with people who are seeking it, covering topics ranging from science and technology to culture and personal experience. Quora also operates Poe, an AI chat platform that provides access to a wide range of AI models and bots.

## Quora Ads Conversion API

The Quora Ads Conversion API (https://www.quora.com/ads/conversion_api_doc) is a server-to-server integration that allows advertisers to send event data — including website events, app installs, and offline conversions — directly to Quora Ads Manager. By bypassing browser-based tracking limitations, the Conversion API improves conversion match rates, which can increase return on ad spend (ROAS) and lower cost per action (CPA). Advertisers authenticate using a token generated in the Conversion API tab of Quora Ads Manager, and requests must include the token, ad account ID, and the Quora click ID (qclid) captured when users interact with ads.

## Poe API

The Poe API (https://creator.poe.com/) is a developer platform that provides access to hundreds of AI models and bots through a single OpenAI-compatible interface. Developed by Quora, Poe enables developers to integrate frontier AI models from major labs — including Claude, GPT-4, Gemini, and many open-source models — as well as millions of community-created bots, all through one API key and unified billing. The platform supports text, image, video, and audio generation. Developers can also build and deploy their own server bots using the fastapi-poe Python library and the Poe Protocol specification. The API is rate-limited to 500 requests per minute, and authentication uses a Bearer token or OpenAI-compatible API key header obtained from https://poe.com/api/keys.
